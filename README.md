## 📖 Sobre o projeto

O **Sistema de Gestão de Condomínios** foi desenvolvido para modernizar a administração de condomínios, oferecendo uma plataforma simples, intuitiva e segura para moradores e administradores.

O sistema permite realizar reservas de áreas comuns, gerenciar moradores, acompanhar eventos e controlar informações importantes do condomínio em um único lugar.

## ⚙️ Instalação

Clone o repositório

```bash
git clone https://github.com/lukasousarj021/Sistema-de-gestao-de-condominios.git
```

Entre na pasta

```bash
cd Sistema-de-gestao-de-condominios
```

Instale as dependências

```bash
npm install
```

Execute o projeto

```bash
npm run dev
```
